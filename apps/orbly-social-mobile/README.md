# Orbly Social Mobile

Bağımsız Expo uygulaması. `@orbly/*` kodu `packages/` altında; marka dosyaları `branding/` altında. Üst repo veya web/API klasörlerinden import veya script fallback yok.

```bash
yarn install
yarn dev
```

Canlı yayın (LiveKit) için development build: `yarn prebuild` → `yarn ios` / `yarn android`.
